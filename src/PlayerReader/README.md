# PlayerReader 读取玩家存档
- 作者: 少司命
- 出处: 无
- 读取`xxx.plr`文件的人物数据保存至tshock

## 说明
该插件会在TS启动目录生成一个`PlayerReaders`文件夹，请把需要转换的存档文件放置在此文件夹。

## 指令

| 语法                           |  权限                 | 说明                                   |
| ----------------------------- | :-------------------: | :-----------------------------------: |
| /readplayer                   | tshock.readplayer.use | 读取并保存文件夹下所有文件				|
| /readplayer [file_name_]      | tshock.readplayer.use | 读取并保存文件夹下指定文件                |
